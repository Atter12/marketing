Activación de Pixel

Genera el código de tracking para Facebook, TikTok y Google. Configura eventos de conversión y optimiza tus campañas publicitarias.

Configura Facebook Pixel

Encuentra tu Pixel ID en Meta Events Manager

Configura TikTok Pixel

Encuentra tu Pixel ID en TikTok Events Manager

Configura Google Tag

Encuentra tu ID en Google Analytics → Admin → Data Streams
Encuentra tu ID en Google Tag Manager
Encuentra tu ID en Google Ads → Tools → Conversions

Instrucciones de Instalación

  1. 1 Genera el código ingresando tu Pixel ID y seleccionando los eventos que deseas trackear.
  2. 2 Copia el código base y pégalo justo después de la etiqueta <head> en todas tus páginas.
  3. 3 Dispara eventos llamando a las funciones correspondientes cuando ocurra cada acción (clic en botón, envío de formulario, etc.).
  4. 4 Verifica la instalación usando las herramientas de debug de cada plataforma (Meta Pixel Helper, TikTok Pixel Helper, Google Tag Assistant).
