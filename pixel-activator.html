Configura TikTok Pixel

¿Dónde encuentro mi Pixel ID de TikTok?
  1. Ve a TikTok Ads Manager e inicia sesión
  2. En el menú lateral izquierdo, haz clic en "Assets" (Recursos)
  3. Luego selecciona "Events" (Eventos)
  4. Verás una lista de tus pixels. Haz clic en el pixel que quieres usar
  5. En la página del pixel, busca la sección "Pixel Code" o "Pixel ID"
  6. El Pixel ID tiene formato: C1234567890ABCDEF (empieza con "C" seguido de letras y números)
  7. Copia ese ID completo y pégalo en el campo de arriba
💡 Tip: Si no tienes un pixel, puedes crear uno nuevo desde TikTok Events Manager → "Create Pixel"

¿Cómo pedir el pixel a mi supervisor?

Si no tienes acceso a TikTok Ads Manager, pide a tu supervisor o al administrador de la cuenta:

  1. Acceso a Events Manager en TikTok Events Manager (o que cree el pixel y te comparta el Pixel ID).
  2. El Pixel ID tiene formato tipo C1234567890ABCDEF (empieza con "C").
  3. Para pruebas pueden crear un pixel de prueba o darte un ID de desarrollo.

Ejemplo de mensaje: "Hola, necesito el Pixel ID de TikTok (Events Manager) para implementar el tracking en la web. ¿Puedes compartirlo o darme acceso a Events Manager para copiarlo?"

Instrucciones de Instalación

  1. 1 Genera el código ingresando tu Pixel ID y seleccionando los eventos que deseas trackear.
  2. 2 Copia el código base y pégalo justo después de la etiqueta <head> en todas tus páginas.
  3. 3 Dispara eventos llamando a las funciones correspondientes cuando ocurra cada acción (clic en botón, envío de formulario, etc.).
  4. 4 Verifica la instalación con TikTok Pixel Helper (Chrome) o la pestaña "Test Events" en TikTok Events Manager.

¿Qué significa esto?

Cuando instalas el TikTok Pixel en tu sitio, cada acción (ver producto, agregar al carrito, comprar) se envía a TikTok. Así puedes:

  • Optimizar anuncios: TikTok encuentra más usuarios similares a quienes compraron
  • Medir conversiones: Sabes cuántas ventas vienen de cada campaña
  • Audiencias y retargeting: Mostrar anuncios a quienes visitaron tu web o abandonaron el carrito

Simular Eventos de Conversión

Haz clic en cada evento para simularlo y verificar que se envíe correctamente. Estos son los eventos más importantes para tu negocio:

Cuándo usar: En páginas de producto. TikTok usa este evento para optimizar y audiencias.
Cuándo usar: Al hacer clic en "Agregar al carrito". Ideal para audiencias de carrito abandonado.
Cuándo usar: Cuando el usuario inicia el proceso de checkout en TikTok Ads.
Cuándo usar: En la página de confirmación de compra. Evento clave para optimización y ROI.
Cuándo usar: Cuando envían un formulario (contacto, registro, lead). Muy útil para conversiones.

Próximos Pasos para Tu Sitio Web

Nota: La verificación en esta página confirma que el código funciona. Ahora necesitas instalarlo en tu sitio web real.

  1. Copia el código base: Arriba tienes el código completo. Cópialo y pégalo justo después de la etiqueta <head> en todas las páginas de tu sitio web.
  2. Implementa eventos específicos: Los comentarios en el código muestran cómo disparar cada evento. Agrega esos códigos JavaScript en las acciones correspondientes:
    • ViewContent → En páginas de producto
    • AddToCart → Al hacer clic en "Agregar al carrito"
    • CompletePayment → En la página de confirmación de compra
    • SubmitForm → Al enviar formularios
  3. Verifica en tu sitio: Usa TikTok Pixel Helper (Chrome) para ver eventos en tiempo real.
  4. Monitorea resultados: En TikTok Events Manager → tu pixel → Test Events o Diagnostics.

Importante: Pega el código base justo después del <head>. Los eventos (ViewContent, CompletePayment, etc.) se disparan con JavaScript en cada acción.
