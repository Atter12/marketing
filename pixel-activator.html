Instrucciones de Instalación

  1. 1 Genera el código ingresando tu Pixel ID y seleccionando los eventos que deseas trackear.
  2. 2 Copia el código base y pégalo justo después de la etiqueta <head> en todas tus páginas.
  3. 3 Dispara eventos llamando a las funciones correspondientes cuando ocurra cada acción (clic en botón, envío de formulario, etc.).
  4. 4 Verifica la instalación usando las herramientas de debug de cada plataforma (Meta Pixel Helper, TikTok Pixel Helper, Google Tag Assistant).

¿Qué significa esto?

Cuando instalas el pixel en tu sitio web, cada acción de tus visitantes (ver producto, agregar al carrito, comprar) se envía automáticamente a Facebook/TikTok/Google. Esto permite a estas plataformas:

  • Optimizar tus anuncios: Encuentran más personas similares a quienes compraron
  • Medir resultados: Sabes exactamente cuántas ventas vienen de cada campaña
  • Crear audiencias: Puedes mostrar anuncios a quienes visitaron tu sitio
  • Retargeting: Recordar a quienes abandonaron el carrito

Simular Eventos de Conversión

Haz clic en cada evento para simularlo y verificar que se envíe correctamente. Estos son los eventos más importantes para tu negocio:

Cuándo usar: Cuando alguien ve una página de producto. Ayuda a Facebook a entender qué productos interesan a tus visitantes.
Cuándo usar: Cuando alguien agrega un producto al carrito. Permite crear audiencias de "carrito abandonado" para retargeting.
Cuándo usar: Cuando alguien inicia el proceso de pago. Ayuda a optimizar campañas para conversiones de checkout.
Cuándo usar: Cuando se completa una compra. Es el evento más importante para optimizar campañas y medir ROI.
Cuándo usar: Cuando alguien completa un formulario de contacto, suscripción o descarga. Ideal para negocios B2B o servicios.

Próximos Pasos

  1. Instala el código en tu sitio: Copia el código generado arriba y pégalo en el <head> de todas tus páginas
  2. Verifica con Meta Pixel Helper: Instala la extensión de Chrome para ver eventos en tiempo real
  3. Revisa en Events Manager: Ve a Meta Events Manager → Test Events para verificar que los eventos lleguen correctamente
  4. Implementa eventos en tu sitio: Agrega el código JavaScript de cada evento en las acciones correspondientes (botón de compra, formulario, etc.)

Importante: El código base debe ir en el <head> de tu página. Los eventos específicos se disparan con JavaScript cuando ocurre la acción.
